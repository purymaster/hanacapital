# 하나캐피탈 자산관리시스템 구축

## 목차

- [기술스택](#기술스택)
- [설치](#설치)
- [이슈사항](#이슈사항)

## 기술스택

Gulp5, EJS, Sass

## 설치

```bash
npm install gulp -g
npm install gulp-cli -g
npm install
```

## 실행

```bash
gulp
```

## 이슈사항

- 파일 수정 시, 전체 파일을 생성하는 현상(추후 수정)
- gulp-cli 전역 설치시 file already exists 에러 출력될 경우, --force으로 강제 설치 후 진행
